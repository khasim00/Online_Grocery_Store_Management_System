# Online_Grocery_Store_Management_System
Built a fully responsive grocery e-commerce website featuring product filtering, shopping cart, and checkout workflows with server-side validation.   Designed a modern, cross-device compatible UI with optimised page-load performance.   Implemented secure server-side data handling ensuring data integrity across all user interactions.
